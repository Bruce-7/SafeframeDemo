# SafeframeDemo
一个安全密码文本框<br/>
![效果1](https://github.com/Bruce-7/SafeframeDemo/blob/master/SafeframeDemo/images/safeframe.gif)<br/>
一个简单的自定义密码文本框和随机键盘,可适用于纯代码,xib,storyboard等情况.<br/><br/>

可以使用Pod install<br/>
platform :ios, '7.0'<br/>

target :'your project' do<br/>

pod 'Safeframe'<br/>

end  <br/>
